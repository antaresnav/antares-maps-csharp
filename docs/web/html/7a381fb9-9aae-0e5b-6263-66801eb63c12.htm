UiSettingsDoubleTapGesturesEnabled Property

Gets whether double tap gestures are enabled/disabled.

Sets the preference for whether double tap gestures should be enabled or disabled. If enabled, users can zoom the map with a double tap.If disabled, double tap has no effect.This setting doesn't restrict programmatic movement and animation of the camera.

The default value is true.

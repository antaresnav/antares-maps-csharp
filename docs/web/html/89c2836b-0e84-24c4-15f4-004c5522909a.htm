LatLngDistanceTo Method

Calculates the geodetic distance between this point and the given point according to the ellipsoid model of WGS84.Altitude is neglected from calculations.

The implementation shall calculate this as exactly as it can.However, it is required that the result is within 0.35% of the correct result.

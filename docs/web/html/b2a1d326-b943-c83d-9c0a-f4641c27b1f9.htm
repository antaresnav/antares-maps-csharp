BearingTo
Calculates the azimuth between this point and the given point according to the ellipsoid model of WGS84.The azimuth is relative to true north. The Coordinates object on which this method is called is considered the origin for the calculation and the Coordinates object passed as a parameter is the destination which the azimuth is calculated to. When the origin is the North pole and the destination is not the North pole, this method returns 180.0. When the origin is the South pole and the destination is not the South pole, this method returns 0.0. If the origin is equal to the destination, this method returns 0.0. The implementation shall calculate the result as exactly as it can. However, it is required that the result is within 1 degree of the correct result.
DistanceTo
Calculates the geodetic distance between this point and the given point according to the ellipsoid model of WGS84.Altitude is neglected from calculations.

The implementation shall calculate this as exactly as it can.However, it is required that the result is within 0.35% of the correct result.

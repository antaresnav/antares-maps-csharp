AntaresMaps.Map Namespace

This namespace defines the AntaresMaps base classes.
Classes
  ClassDescription
AntaresMapOptions
Defines configuration AntaresMapOptions for a AntaresMap. These options can be used when adding a map to your application programmatically.
CameraMoveStartedEventArgs
Provides data for the CameraMoveStarted event.
CameraUpdateFactory
A class containing methods for creating CameraUpdate objects that change a map's camera. To modify the map's camera, call AnimateCamera(CameraUpdate), AnimateCamera(CameraUpdate, AntaresMap.CancelableCallback) or MoveCamera(CameraUpdate), using a CameraUpdate object created with this class.

For example, to zoom in on a map, you can use the following code:

AntaresMap map = ...; map.AnimateCamera(CameraUpdateFactory.ZoomIn());

Prior to using any methods from this class, you must wait for an AntaresMap to become available from a MapFragment or MapView that you have added to your application.You can obtain the AntaresMap instance by calling getMapAsync() and waiting for the onMapReady(AntaresMap map) callback.

MapView

Projection
A projection is used to translate between on screen location and geographic coordinates on the surface of the Earth(LatLng). Screen location is in screen pixels(not display pixels) with respect to the top left corner of the map(and not necessarily of the whole screen)..
UiSettings
Settings for the user interface of an AntaresMap. To obtain this interface, call GetUiSettings().

ICameraUpdate
Defines a camera move. An object of this type can be used to modify a map's camera by calling animateCamera(CameraUpdate), animateCamera(CameraUpdate, AntaresMap.CancelableCallback) or moveCamera(CameraUpdate). To obtain a CameraUpdate use the factory class CameraUpdateFactory. For example, to zoom in on a map, you can use the following code: AntaresMap map = ...; map.animateCamera(CameraUpdateFactory.zoomIn());
